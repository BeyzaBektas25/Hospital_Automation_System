# 🏥 Hospital Management System (C++)
![C++](https://img.shields.io/badge/Language-C%2B%2B-blue)
![OOP](https://img.shields.io/badge/OOP-Object--Oriented-success)
![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey)

This project is a **console-based Hospital Management System** developed in **C++**. It was designed using **Object-Oriented Programming (OOP)** principles to manage patient records and keep track of admitted and discharged patients.

## 🚀 Features

* **Patient Registration:** Stores each patient's number, name, surname, blood type, and phone number.
* **Phone Number Validation:** Ensures that the phone number contains exactly **10 digits** to prevent invalid data entry.
* **Patient Discharge:** Searches for a patient by their patient number, removes them from the current patient list, and adds them to the discharged patient list.
* **Patient Lists:** Displays detailed information about all current patients and a list of discharged patients.
* **Memory-Safe Implementation:** Uses controlled array operations and optimized loops to help prevent buffer overflow and out-of-bounds access.

## 🛠️ Technologies Used

* **Language:** C++
* **Libraries:** `<iostream>`, `<string>`, `<windows.h>`

## 💻 How to Run

1. Clone or download this repository.
2. Open the `main.cpp` file in any C++ IDE or compiler (such as Dev-C++, Code::Blocks, Visual Studio, or VS Code).
3. Build and run the project.
